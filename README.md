express-angular-seed
====================

once you have a new repo set remote with

```git remote set-url origin [repo address]```

## To start dev

nodemon bin/www
redis-server
sass --watch scss:public/stylesheets/

## Features

* Report page
 * Over performing
 * netural
 * Under performing
 * Grid and list view
 * download report
* Detail page
 * visits over time
